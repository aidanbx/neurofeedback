# Programs

Each subdirectory is a self-contained IronBCI program.

## Directory structure

```
programs/
  README.md             — this file
  default_report.py     — fallback analysis run for any session with no program-specific report
  <program-id>/
    manifest.json       — program metadata (id, title, description, version)
    program.js          — self-contained JS program class
    report.py           — optional; if present, run instead of default_report.py
```

---

## Creating a new program

### 1. `manifest.json`

```json
{
  "id": "my_program",
  "title": "My Program",
  "description": "One-sentence description.",
  "version": "1.0"
}
```

### 2. `program.js`

Register a class on `window.NFPrograms`:

```js
'use strict';
(function () {

class MyProgram {
  constructor(host) {
    // host = {
    //   container       — DOM element to render into
    //   audioCtx        — Web Audio context (null until session.start())
    //   masterGain      — GainNode connected to destination
    //   session: {
    //     start()       — POST /api/training/start, triggers startSession()
    //     stop()        — POST /api/training/stop, triggers stopSession()
    //     log(obj)      — append one line to program_metrics.jsonl
    //     active        — boolean
    //   }
    //   onCalibrated()           — optional; tell host calibration finished
    //   onCalibrationProgress()  — optional; update host status text
    // }
    this._host = host;
    this._buildDOM();
  }

  updateHost({ audioCtx, masterGain }) {
    // Called just before startSession() with the live AudioContext.
    this._host.audioCtx   = audioCtx;
    this._host.masterGain = masterGain;
  }

  startCalibration() { /* optional; call host.onCalibrated() when done */ }
  startSession()     { /* called after host.session.start() */ }
  stopSession()      { /* called after host.session.stop()  */ }
  destroy()          { this._host.container.innerHTML = ''; }

  onTick(metrics, elapsed, appState) {
    // metrics   — band powers, quality score, diagnostics
    // elapsed   — wall-clock seconds (Date.now() / 1000)
    // appState  — full /api/state response + { channel: selectedChannel }
  }

  _buildDOM() { /* build all UI inside this._host.container */ }
}

window.NFPrograms = window.NFPrograms || {};
window.NFPrograms.my_program = MyProgram;

})();
```

**Rules:**
- The program owns everything inside `host.container`. Build your own buttons, canvases, sliders.
- Call `host.session.start()` to begin recording (user gesture required for AudioContext).
- Call `host.session.log({ type: 'event', ... })` to append structured data to `program_metrics.jsonl`.
- No calibration needed? Call `host.onCalibrated()` immediately in `startCalibration()`, or skip calibration entirely.
- Use CSS classes from `static/training/programs.css` for the standard two-column layout (`.program-body`, `.program-main`, `.program-sidebar`, `.nf-section`, `.nf-stat`, etc.).

### 3. `report.py` (optional)

If present, this script is run automatically when the session ends, instead of `default_report.py`.

```python
#!/usr/bin/env python3
"""
Custom report for my_program.
Usage:  python report.py <session_dir>
Output: <session_dir>/report.html
"""
import sys
from pathlib import Path

# You can import shared utilities from the programs directory if needed.
# sys.path.insert(0, str(Path(__file__).parent.parent))
# from report_utils import load_session, html_page, band_power_section

session_dir = Path(sys.argv[1])

# Read standard session files
# derived_metrics.csv  — per-tick band powers, quality, diagnostics
# metadata.json        — recording info, eye/posture/clench events
# program_metrics.jsonl — custom events logged by program.js via session.log()

html = "<html>...</html>"
(session_dir / "report.html").write_text(html)
```

If no `report.py` is found, `default_report.py` runs (general EEG quality report).

## Session folder layout

After a session ends the folder at `sessions/<timestamp>/` contains:

| File | Contents |
|------|----------|
| `metadata.json` | device, timestamps, eye/posture/clench/blink events, training program info |
| `raw.csv` | raw ADC samples, all channels |
| `derived_metrics.csv` | per-tick band powers, quality score, artifact metrics |
| `spectrogram.csv` | time-frequency power matrix |
| `program_metrics.jsonl` | newline-delimited JSON logged by `session.log()` |
| `report.html` | self-contained analysis report (generated by report.py) |
